# To-Do API con FastAPI

API sencilla para gestión de tareas (To-Do List), desarrollada con **FastAPI** y diseñada para ser desplegada en **Render** utilizando una base de datos en **Supabase**.

## Características (Features):

- **Framework:** FastAPI para una API rápida y moderna.
- **Despliegue:** Optimizado para la plataforma Render.
- **Base de datos:** Integración con Supabase (en desarrollo).

## Instalación

1. **Clona el repositorio:**
   ```
   git clone <URL_DE_TU_REPO>
   cd <NOMBRE_DE_CARPETA>
    ```
2. Entorno virtual (Conda):
    ```
    conda create -n p-fastapi-1 python=3.12
    conda activate p-fastapi-1
    ```
3. Instalar dependencias:
    ```
    pip install -r requirements.txt
    ```

## Uso

Para iniciar el servidor en modo desarrollo:
    ```
    uvicorn app.main:app --reload
    ```
